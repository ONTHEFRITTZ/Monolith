A low cost EVM bridge and off ramp for the Monad ecosystem.
=======

Low-cost USDC ⇄ MON bridge targeting the Monad ecosystem with smart-account UX primitives.

## Folder Layout

- `apps/` – frontend (`web`) and backend (`api`) applications.
- `packages/` – shared libraries (design system, SDK, config).
- `docs/` – project outline, specs, and status notes.

## Tooling

- **Package manager:** npm workspaces.
- **Task runner:** Turborepo (`turbo.json`).
- **Code quality:** ESLint, Prettier, TypeScript, Husky pre-commit hook with lint-staged.

## Getting Started

```bash
npm install
npm run dev
```

Add your environment variables in `.env` files as you scaffold services.
